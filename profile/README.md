# Audio Timing Protocol (ATP)

ATP is an open-source protocol for streaming audio from a host to multiple clients
with sample-accurate synchronization.

## Goals
- **Simple** — easy to implement in any language
- **Accurate** — sample-accurate playback via clock synchronization
- **Extensible** — versioned protocol
